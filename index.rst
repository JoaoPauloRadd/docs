-------------

JPHP is a new implementation for PHP which uses the Java VM. It supports many features of the PHP language (5.4+).
How does it work? JPHP is a compiler like ``javac``, it compiles php sources to JVM bytecode and then can execute the result on the Java VM.

.. note::

    Supports: JDK 1.6+ (OpenJDK, Oracle), Linux, Windows, MacOS, etc.


-------------

JPHP это с нуля написанный движок для PHP, который использует Java VM. Движок поддерживает множество возможностей PHP (5.4+).
Как это работает? JPHP является компиляторов по типу ``javac``, он компилирует php код в JVM байткод, а затем выполняет
полученный результат на Java VM.

.. note::

    Поддерживается: JDK 1.6+ (Oracle или OpenJDK), Linux, Windows, MacOS и т.д.
